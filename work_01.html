오토북스 웹 프로그램

Design 100% | Markup 100%

세무·회계 웹 프로그램 신규 런칭 예정 웹사이트. Vue 컴포넌트 기반으로 제작하여 팀 협업과 유연한 수정이 가능하도록 설계했습니다. 방대한 데이터를 효율적으로 처리할 수 있는 테이블 UI에 집중하고, 사용자가 한눈에 자료를 파악할 수 있도록 직관적이고 명확한 화면 구조를 구현했습니다. 내용이 많아도 쉽게 이해할 수 있도록 시각적 흐름과 정보 계층을 세심하게 설계했습니다.

  • 기존 프로그램은 방대한 데이터를 구조 없이 한 화면에 나열했으며, 일관성 없이 사용된 다양한 색상들이 시각적 혼란을 야기하였습니다.

  • 새롭게 런칭할 프로그램은 필요성을 구분하여 데이터의 위계를 체계적으로 구성하여 꼭 필요한 데이터만 한눈에 들어오도록 시각적 피로도를 줄였고, 사용자가 필요한 정보를 더 쉽게 찾을 수 있도록 필터를 세분화하였습니다.
    또한 최대 1,000개의 데이터를 한 페이지에서 확인할 수 있을 정도로 방대한 정보량을 고려하여, 테이블 상단을 고정해 스크롤 시에도 항목을 쉽게 인식할 수 있도록 편의성을 강화했습니다.
